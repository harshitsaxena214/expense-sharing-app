# 💸 Settlix – Simplify Group Expenses

**Settlix** is a modern expense-splitting web app that helps you track, split, and settle expenses with friends — without the usual confusion.

🔗 **Live App:** https://settlix.vercel.app

---

## ✨ Features

- 💰 Split expenses easily with friends or groups  
- ⚡ Fast and responsive UI  
- 🧠 Automatic calculation of who owes whom  
- 📊 Clear breakdown of balances  
- 🔄 Real-time updates *(if implemented)*  
- 📱 Clean and minimal design  

---

## 🚀 Tech Stack

- **Frontend:** Next.js  
- **Backend:** Express.js  
- **Styling:** Tailwind CSS
- **Deployment:** Vercel  

---

## 📌 How It Works

1. Create a group  
2. Add expenses  
3. Select who paid  
4. Choose participants  
5. Settlix calculates balances automatically  
6. Settle up without confusion 🎉
